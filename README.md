# ImgConverter

Утилита для конвертации изображений с поддержкой:
- Смены формата (png, webp, jpg, jpeg)
- Изменения размера
- Черно-белого режима
- Настройки качества

## Опции

### Обязательные аргументы
- `file`: Путь к входному файлу изображения

# Необязательные опции
- `-m, --mode`: Формат конвертации
  - Варианты: png, webp, jpg, jpeg
  - По умолчанию: jpg

- `-q, --quality`: Качество сжатия 
  - Диапазон: 0-100%
  - По умолчанию: 100%

- `-o, --optimize`: Включить оптимизацию изображения
  - Флаг без значения
  
- `-s, --size`: Изменение размера изображения
  - Формат: ШИРИНАxВЫСОТА (например, 800x600)

- `-w, --white`: Фон при конвертации
  - Белый фон для конвертации из png/webp
  - По умолчанию: черный фон

- `-g, --grey`: Преобразование в черно-белый режим


## Использование

```bash
# Базовое использование
imgconverter image.png

# Конвертация в webp с качеством 80%
imgconverter image.png -m webp -q 80

# Изменение размера
imgconverter image.png -s 800x600

# Черно-белый режим
imgconverter image.png -g

# Полная конвертация png -> jpeg с чернобелым режимом, с размером 800x600  и с белым фоном, вместо прозрачной части
imgconverter image.png -m jpeg -q 80 -s 800x600 -w -g
